# 燃造物 · 愈创木手串文创博客

黑金藏风风格的佛珠手串文创博客，纯静态网站（HTML + CSS + JS），无需构建，开箱即用。

## 内容模块

- **关于木材** — 愈创木等材质介绍
- **木性科普** — 木材科学知识
- **对比鉴别** — 真伪 / 材质对比
- **珠串展示** — 作品图集
- **盘玩养护** — FAQ 与养护指南
- **留言** — 访客留言区

## 技术栈

- 原生 HTML / CSS / JavaScript，零依赖
- 响应式设计，适配移动端
- Google Fonts（Noto Serif SC / Noto Sans SC）
- 自带 logo（logo.png）

## 本地预览

直接用浏览器打开 `index.html` 即可，或在项目目录启动一个静态服务器：

```bash
# Python
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 在线访问

通过 GitHub Pages 免费托管，地址形如：

```
https://<你的用户名>.github.io/ranzaowu-blog/
```

## 文件结构

```
ranzaowu-blog/
├── index.html   # 页面主体（6 个模块）
├── style.css    # 黑金藏风样式

```
